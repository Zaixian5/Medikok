<h3>기능</h3>
1. 약 봉투 사진 촬영 - 복용 약물 목록에 사진 추가<br>
2. 약 검색 - 약을 검색하여 용법, 사진 등 정보 조회(식약처 공개 API 활용)<br>
3. 알람 추가 - 복용 시간, 요일, 복용 약물을 선택하면 알람 카드가 추가 됨. 스위치로 복용 여부 체크<br>
